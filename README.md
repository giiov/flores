# Buquê!
> Uma aplicação web desenvolvida em **React.js** que simula a criação de um buquê virtual. Apenas pela diversão!

---

## Funcionalidades 
* Carrossel para escolher o seu vaso
* Escolha de flores para montar o buquê
* Carrossel para escolher seu cartão
* Campo para escrever uma mensagem e finalizar!

---

## Tecnologias
* React.js
* JavaScript
* HTML5 e CSS3

---

## Como rodar o projeto
```bash 
# Clone o repositório
git clone https://github.com/giiov/flores

# Acesse a pasta
cd flores

# Instale as dependências
npm install

# Rode o projeto
npm run dev

```
